# Ladakh 2026 — Road Trip Website

Single-page React + Tailwind CSS trip planner for a Leh-to-Leh Ladakh road trip, Oct 3–12, 2026.

## Quick Start (local)

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Create a new repo on GitHub named `ladakh-2026`
2. Push this project:
   ```bash
   cd ladakh-2026-site
   git init
   git add -A
   git commit -m "Initial commit — Ladakh 2026 trip site"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/ladakh-2026.git
   git push -u origin main
   ```
3. In GitHub: **Settings → Pages → Source → GitHub Actions**
4. The workflow runs automatically on push. Your site will be live at:
   `https://YOUR_USERNAME.github.io/ladakh-2026/`
